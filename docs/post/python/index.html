# python哲学

输出

  1. 打印(3种输出方法)

    • print("name is :"+name) 通过连接符 + 拼接
    • print("name is : %s" %name) 通过格式符 %s,%d 替换
    • print("name is :{}".format{name}) 通过格式化函数 format{} 格式化

  2. 引号和注释

    • “ ” 双引号 和 ’ ‘ 单引号不区分，但是不能交叉使用
    • 单行注释符号：#
    • 多行注释符号：“”“ contents ”“” (3对双引号)，’’’ contents ‘'' (3对单引号)
